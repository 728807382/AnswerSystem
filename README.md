# AnswerSystem
网上辅导答疑管理系统的设计与实现
1、设计内容
设计一个网上辅导答疑管理系统，该系统的用户分别是：学生和教师。不同的用户拥有不同的权限，各自完成各自的管理功能，不同的用户看到不同的系统功能。按照要求完成所有的功能和模块。
学生的主要功能模块包括：
（1）学生的注册与登录
（2）问题库查询
（3）学生提问
教师的主要功能模块包括：
（1）教师的登录与注册
（2）人工答疑
（3）问题管理
（4）学生管理
2、主要数据库表单
（1）教师基本信息表
主要字段有：教师id、姓名、性别、职称、联系电话
（2）问答信息表
主要字段有：问题id、问题标题、问题描述、提问人、提问时间、回答教师id、回答时间、回答内容
（3）	学生信息表（自建）
